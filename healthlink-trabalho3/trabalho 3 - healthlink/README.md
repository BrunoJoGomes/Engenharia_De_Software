# HealthLink — Trabalho 3: Modelagem

Plataforma de Telemedicina · Engenharia de Software

## Visão Geral

Este repositório contém a modelagem em profundidade de três fatias verticais do sistema **HealthLink**, uma plataforma de telemedicina que conecta pacientes a profissionais de saúde de forma remota.
